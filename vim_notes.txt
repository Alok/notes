VIM NOTES
==================
macro recording/playback:
    qx  start recording (save in register 'x') 
    q   stop recording
    @x  replay the recording from register 'x'

good:
    http://vimcasts.org/

plugins:
    github.com/kien/ctrlp.vim
        let g:ctrlp_extensions = ['tag']
        let g:ctrlp_cmd = 'CtrlPMixed' " CtrlPTag
    ctags.sourceforge.net 
        export PATH="/usr/local/bin:$PATH"
    http://vim.wikia.com/wiki/Omni_completion -or- https://github.com/Shougo/neocomplcache

fundamental:
    capitalizing (shift) performs the "mirror" of the command
    command is multiplied by a prepended number
    command is located by [hjkl]
  
modifiers:
    * search forwards
    # search backwards

window splitting:
    :sp :vsp ctrl-w-[w q o hjkl]

buffer hopping:
    ctrl-6 :ls

file management:
    :E. d D - 

"go" movement:
    gg G ^ $ f t ; % [{

"go" through the change list:
    g; g, :changes

editing:
    i I a A o O J

indentation, (visual mode):
    > <

indentation, (insert mode):
    ctrl-t ctrl-d
